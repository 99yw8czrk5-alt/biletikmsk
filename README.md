# Flight Watch

Монитор дешевых билетов Москва (`MOW`) -> Бангкок (`BKK`) с уведомлениями в Telegram.

## Что уже настроено

- Проверка Aviasales/Travelpayouts API.
- Фильтр по максимальной цене.
- Пометка багажа:
  - `есть`, если API явно отдал багаж;
  - `нет`, если API явно отдал отсутствие багажа;
  - `надо проверить`, если API не дал надежного поля по багажу.
- Защита от повторных уведомлений по одному и тому же билету.
- Режим одной проверки и режим постоянной проверки каждые 6 часов.
- Команда в Telegram: `цена` или `/price`, чтобы получить ближайшие билеты в чат.

## Подготовка

```bash
python3 --version
```

Сторонние библиотеки не нужны.

## Получить Telegram chat id

1. Напиши любое сообщение своему Telegram-боту.
2. Запусти:

```bash
PYTHONPATH=src python3 -m flight_watch.telegram_chat_id
```

3. Скопируй напечатанный `TELEGRAM_CHAT_ID` в `.env`.

## Проверить один раз

```bash
PYTHONPATH=src python3 -m flight_watch.monitor --once
```

## Запустить постоянный мониторинг

```bash
PYTHONPATH=src python3 -m flight_watch.monitor
```

По умолчанию проверка идет каждые 6 часов.

## Запустить Telegram-бота с командой цены

```bash
PYTHONPATH=src python3 -m flight_watch.monitor --bot
```

Когда человек пишет `цена` или `/price`, бот сразу отправляет сообщение `Ищу билеты Москва -> Бангкок...`, а затем присылает найденные варианты.

Запустить в фоне с логом:

```bash
scripts/start_bot.sh
```

Остановить:

```bash
scripts/stop_bot.sh
```

Проверить статус и последние строки лога:

```bash
scripts/status_bot.sh
```

## Автозапуск на этом Mac

Чтобы бот работал без Codex и сам поднимался после перезагрузки:

```bash
scripts/install_autostart.command
```

Скрипт сам обновит `com.flight-watch.bot.plist` под текущую папку проекта, скопирует его в `~/Library/LaunchAgents` и запустит бота через `launchd`.

## Запуск не с твоего компьютера

Для автономной работы 24/7 лучше использовать VPS. Готовая инструкция и `systemd`-сервис лежат в [DEPLOY.md](DEPLOY.md).

Самый простой бесплатный вариант без твоего компьютера — Render Free. Инструкция лежит в [RENDER.md](RENDER.md).

После запуска напиши боту:

```text
цена
```

или:

```text
/price
```

Бот ответит ближайшими билетами, которые проходят текущие настройки `MAX_PRICE`, `DAYS_AHEAD` и `BAGGAGE_MODE`.

Если человек напишет любой другой текст, бот ответит подсказкой: нужно написать `цена` или `/price`, чтобы увидеть билеты Москва -> Бангкок.

## Проверить тесты

```bash
PYTHONPATH=src python3 -m unittest discover -s tests -v
```

## Изменить цену

В `.env` поменяй:

```env
MAX_PRICE=60000
```

## Сделать фильтр багажа жестким

По умолчанию стоит:

```env
BAGGAGE_MODE=allow_unknown
```

Это значит: присылать билеты, где багаж есть или API не дал ясной информации. Если хочешь присылать только билеты с явно найденным багажом:

```env
BAGGAGE_MODE=strict
```
